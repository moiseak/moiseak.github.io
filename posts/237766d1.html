字节跳动青训营(一)

Created2024-09-28|Updated2024-09-29

题目分为三个等级: 简单, 中单, 困难. 我做完觉得不错的题会放上来

简单题

1.x到y的最少步数

我们假设每一步对应到每一个位置。从一个整数位置 x 走到另外一个整数位置 y，每一步的长度是正整数，每步的值等于上一步的值 -1， +0，+1。求 x 到 y 最少走几步。并且第一步必须是 1，最后一步必须是 1，从 x 到 y 最少需要多少步。

这题很有意思, 非常能展现贪心算法的魅力.
因为要求最少的步骤, 所以这道题我们使用贪心. 即我们只有+1, -1 两个选择, 因为+0只会浪费步骤, 无法让我们更靠近y.
那么我们就可以一直+1, 直到current_distance超过距离d, 这时我们需要判断这两个的差值是奇数还是偶数.
如果是偶数, 我们就可以通过减少前面+1的次数, 增加-1的次数, 所改变的每一步与原来的情况都会有2的差值, 就这么调整下去, 在不改变步数的情况下, 偶数差值可以完美抵消.
而奇数差值无法按照这个方法抵消, 因为无法使用+0, 我们每次调整改变的差值必定是2. 所以需要继续+1, 直到二者的差值为偶数, 可以完美消除差值.

2.创意标题匹配

线上服务的时候，会根据用户的搜索词触发的 bidword 对创意中的通配符（通配符是用成对 {} 括起来的字符串，可以包含 0 个或者多个字符）进行替换 ，用来提升广告投放体验。例如：“{末日血战} 上线送 SSR 英雄，三天集齐无敌阵容！”，会被替换成“帝国时代游戏下载上线送 SSR 英雄，三天集齐无敌阵容！”。给定一个含有通配符的创意和一句标题，判断这句标题是否从该创意替换生成的。

这道题的意思是我们会给定一个字符串, 其中含有"{}"包裹起来的部分, 这部分可以被替换为其他字符, 但括号外的字符必须完美匹配.
输入样例
4
ad{xyz}cdc{y}f{x}e

adcdcefdfeffe
adcdcefdfeff
dcdcefdfeffe
adcdcfe

输出样例

True
False
False
True

这道题的核心就是将我们定义的可以表示其他字符的{xxx}这部分替换为在正则表达式中可以表示任何字符的.*, 之后利用正则表达式的判断就可以.
re.sub(r'{[^}]*}', '.*', template)中sub函数表示将第三个参数中的第一个参数替换为第二个参数.
r 表示 “raw string”（原始字符串）。在 Python 中，通常会在正则表达式前面加上 r，表示这是一个原始字符串。这样做的目的是让字符串中的反斜杠（\）保持原义，不需要转义。例如，“\n” 在普通字符串中表示换行符，但在原始字符串中它仅表示字符 \ 和 n。
大括号 { 在正则表达式中有特殊含义（表示重复次数），因此如果要匹配实际的大括号字符，必须进行转义。{ 在这里表示匹配左大括号字符 {。
方括号 [] 用来表示字符类，也就是说它匹配的是任意属于方括号中的字符。
^ 在字符类中表示 “非”，即排除字符类中的内容。所以 [^}] 表示 “匹配任何不是右大括号 } 的字符”。
例如，[^}] 会匹配字符 a、1、*，但不会匹配右大括号 }。
星号 * 是一个量词，表示前面的模式可以出现零次或多次。
在这个例子中，它修饰的是 [^}]，即表示 “匹配任意数量（包括零个）不是右大括号 } 的字符”。
这里的右大括号 } 表示匹配实际的右大括号字符 }。

中等题

1.简单四则计算

实现一个基本的计算器来计算一个简单的字符串表达式的值。注意事项如下：

  • 输入是一个字符串表达式（可以假设所给定的表达式都是有效的）

  • 字符串表达式可以包含的运算符号为：左括号 (, 右括号 ), 加号 +, 减号 -

  • 可以包含的数字为：非负整数（< 10）

  • 字符串中不包含空格

  • 处理除法 case 的时候，可以直接省略小数部分结果，只保留整数部分参与后续运算

  • 请不要使用内置的库函数 eval

我们小小的改动一下加点难度: 可以包含的数字为：非负整数（可大于10）

这道题我感觉说是没什么好说的, 看代码基本都能看懂, 这种题就是你做过的话思路就很清晰, 没做过的话就可能有点懵, 所以还是很有做的价值的.
